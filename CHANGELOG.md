# Wallos-Remastered 变更记录

## 说明

本文件记录 `Wallos-Remastered` 分支已经落地的重制能力摘要，不再沿用上游完整英文发布流水。

## 2026-04-28

- 缓存治理：Service Worker 缓存版本升级到 v18，版本化 CSS/JS 改为网络优先并回落到精确缓存，减少旧静态资源导致的“某台设备异常”。
- 缓存治理：订阅私有媒体目录被明确排除出 Service Worker 图片缓存，继续只通过鉴权端点访问，避免已删图或跨账号媒体在浏览器缓存里残留。
- 回归检查：新增 `tests/e2e/client_cache_smoke.mjs` 普通用户浏览器级缓存刷新 E2E，验证客户端缓存刷新提示会持续显示到手动关闭，并检查右下角提示不会撑宽页面；`npm run e2e` 已纳入该检查。
- 回归检查：新增 `tests/e2e/subscription_images_smoke.mjs` 浏览器级订阅图片 E2E，真实上传测试 PNG，验证未压缩上传时原图字节透传、预览图/缩略图不大于原图、图片尺寸文案没有 `[Translation Missing]`、未登录访问媒体端点会被拒绝，并在永久删除临时订阅后确认媒体记录不再可读。
- 回归检查：`npm run e2e` 现在会依次运行订阅页 E2E、订阅图片 E2E 和管理员后台 E2E；管理员后台 E2E 仍要求显式提供未过期管理员凭据，否则安全跳过。
- 管理后台：订阅图片审计补齐闭环，新增“清理孤儿图片”维护动作，可删除数据库不再引用的订阅图片孤儿文件，并显示删除数量、删除体积和失败样本。
- 订阅图片：补齐前端图片变体文案，避免待上传图片卡片的本地文件大小摘要出现 `[Translation Missing]`。

## 2026-04-24

- 回归检查：扩展 `tests/regression_runner.php`，新增静态契约检查，覆盖默认紫色主题、订阅页关键 DOM、订阅页脚本加载顺序、共享请求层、API key 传输约定和订阅图片大小显示槽位。
- 文档：新增 `下一步任务.md`，记录后续自动化回归、订阅页生命周期收敛、会话契约统一、可观测性和 SQLite 友好优化方向。

## 2026-04-25

- 订阅图片：修复未勾选压缩时原图仍被 GD 重新编码导致 PNG 文件膨胀的问题；现在未压缩上传会透传保存原始文件字节，预览图/缩略图仍会经过服务端校验与派生生成。
- 订阅图片：当生成出的预览图或缩略图没有降尺寸且比原图更大时，会复用原图作为该派生版本，避免“预览图比原图大”的反向压缩。
- 会话安全：页面后台挂起超过 30 分钟后回到前台，会提示刷新页面以重载 CSRF 安全令牌；后端返回 `invalid_csrf` 时也会显示同一套说明，避免只看到生硬的英文错误。
- 会话安全：页面底部部署信息区域新增本次页面 CSRF 安全令牌短指纹与 30 分钟建议刷新时间，便于排查页面挂起后的令牌状态；完整令牌仍保持隐藏，避免截图泄露。
- 会话安全：页脚安全令牌时间改为明确跟随当前账号时区，并显示完整时区标识与 UTC 偏移，避免 `CST` 等缩写产生歧义。
- 会话安全：CSRF 页面安全令牌从“前端建议刷新”升级为服务端强制 30 分钟 TTL；过期令牌提交会返回 `invalid_csrf`，需要刷新页面重新加载令牌。
- 回归检查：登录态 smoke test 支持通过测试账号自动登录；可选 `--mutating-auth-checks` 会验证创建、编辑、记录支付、移入回收站和彻底删除临时订阅。
- 回归检查：新增 `tests/e2e/subscriptions_smoke.mjs` 浏览器级订阅页 E2E，用于覆盖分页、三点菜单、编辑、新增保存、实际支付记录、图片预览、列数切换和动态壁纸下按钮可点击性。
- 缓存治理：Service Worker 缓存版本升级，新增管理员清理当前浏览器缓存与发布客户端缓存刷新提示能力。
- 运维维护：管理员后台新增维护入口，可查看日志保留策略、扫描订阅图片缺失派生图和孤儿文件，并手动执行 SQLite `PRAGMA optimize` / `ANALYZE` / `VACUUM`。
- 订阅图片：添加上传前本地文件大小提示；图片预览中展示缩略图、预览图、原图三层服务器文件大小，并标记复用原图的派生层。
- 订阅图片：修复“预览图尺寸已缩小但文件仍大于原图”时不会复用原图的问题；现在只要预览图/缩略图派生文件不小于原图，就直接复用原图，并在管理员维护入口提供一键修复历史过大派生图。

## 2026-04-26

- 会话安全：CSRF 安全令牌过期或后端返回 `invalid_csrf` 时，右下角提醒改为持续显示，只有用户手动关闭或刷新页面才会消失，避免几秒后自动滑走导致看漏。
- 回归检查：强化 `tests/e2e/subscriptions_smoke.mjs` 浏览器级订阅页 E2E，新增分步骤输出、失败截图/HTML/JSON 诊断、前端运行时错误捕获、记录支付弹窗、显示偏好恢复和 CSRF 持久提醒检查。
- 回归检查：新增 `npm run e2e` 统一入口，并把浏览器 E2E 覆盖范围纳入静态契约检查，避免后续删弱关键交互用例。
- 管理后台：安全设置区域新增“运行可观测性”诊断面板，展示最近异常类型分布、客户端缓存刷新提示时间、最新异常流，并支持一键筛选前端异常与请求失败。
- 安全：访问日志和安全异常浏览器改为对日志内容做 HTML 转义，避免异常消息、请求路径或 User-Agent 污染管理员页面。
- 管理后台：维护任务区域新增存储占用概览，可查看数据库、上传目录、订阅图片目录、备份目录、访问日志、安全异常和速率限制用量增长情况。
- 管理后台：订阅图片审计现在统计孤儿文件总体积和明细，并支持一键导出审计 CSV。
- 管理后台：SQLite 维护结果现在显示维护前后数据库大小、页数、空闲页和耗时，便于评估 `VACUUM` 是否有效。
